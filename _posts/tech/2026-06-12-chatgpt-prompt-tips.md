## なぜ「そのまま」の指示ではダメなのか

ChatGPTに「このコード直して」と投げるだけでは、AIは「とりあえず動くコード」を返すだけです。それでは、その場しのぎにしかならず、あなたのスキルは一向に伸びません。

エンジニアに必要なのは、**「なぜそのコードが書かれたのか」という思考プロセス**です。

![AIメンターによるコードレビューのイメージ](/assets/images/ai-mentoring.png)

## AIを「専属メンター」にするための型

AIの回答精度を最大化するには、「役割（ロール）」と「手順（ステップ）」を明確に指定するのがコツです。以下のテンプレートをコピーして使ってみてください。

```text
# あなたの役割
あなたはプロのPythonエンジニアです。プログラミング初心者の私に、分かりやすく技術的なアドバイスをください。

# お願いしたいこと
以下のコードの改善点と、なぜそうすべきかの理由を教えてください。

# コード
x = 10; y = 20; z = x + y; print(z)

```

## 実践：エラー解決が爆速になるプロンプト

エラーで詰まった時は、状況を正確に伝えることが重要です。以下の「指示の型」を保存しておきましょう。

```text
# 状況
Pythonでコードを動かそうとしてエラーが出ました。

# エラーメッセージ
[ここにエラー内容を貼り付け]

# 現在のコード
[ここにコードを貼り付け]

# 指示
1. なぜエラーが起きたのか、初心者にもわかるように解説してください。
2. 修正したコードを提示してください。
3. 今後同じミスを防ぐためのアドバイスをください。

```
